# Gesture Based Volume Controller

## Project Description
This project controls the system volume using hand gestures detected through a webcam. 
It uses MediaPipe for hand tracking and OpenCV for computer vision processing.

## Technologies Used
- Python
- OpenCV
- MediaPipe
- Pycaw (for volume control)

## Requirements
Install the following libraries:

pip install opencv-python
pip install mediapipe
pip install pycaw
pip install numpy

## How to Run the Project

1. Clone the repository
git clone https://github.com/syashita657-web/Gesture-based-volume-controller

2. Open the project folder

3. Run the main Python file
python volume_control.py

4. Allow webcam access.

5. Use hand gestures:
- Increase distance between fingers → Volume Up
- Decrease distance → Volume Down
